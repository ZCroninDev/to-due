# to-due

To-do list web application

## Description

To-due is web application with a to-do list, task tracker, and productivity web appplication.    

## Getting Started

### Dependencies

* HTML5, CSS3, Materialize,  vanilla javaScript

### Installing

* https://materializecss.com/getting-started.html

### Executing program

* I used Visual Studio Code as my code editor 
* Materialize can be installed by npm 

```
npm install materialize-css@next

```
